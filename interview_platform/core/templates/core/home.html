Welcome!

This is the Crazy Powerful Interview Platform!! The most powerful of interview tools to ever grace the internet!

CPIP is an interview platform built for UNIHACK 2025, made to give both interviewer and interviewee an easier time connecting and sharing ideas. This platform uses Django for the web framework and HTML and JavaScript for frontend UI. Websockets, Redis, and Daphne are used to support the real-time updates for the code editor and whiteboard.
